# Eclipse AI 🌑

<img src="https://github.com/user-attachments/assets/23c29f76-6152-467b-b898-b0226f75e5a7" width="250" align="left">
Eclipse AI é uma inteligência artificial para terminal Linux desenvolvida em Python e integrada ao Ollama. O projeto oferece uma experiência simples e leve de conversa com modelos de IA locais, diretamente pelo terminal, sem necessidade de navegador ou serviços em nuvem.

# Recursos
Conversa com modelos locais usando Ollama
Interface colorida no terminal
Arte ASCII personalizada na inicialização
Personalização do nome do usuário
Funcionamento offline
Código simples e fácil de modificar
Baixo consumo de recursos
Tecnologias Utilizadas
Python 3
Ollama
Modelo Qwen 2.5
Terminal Linux
Objetivo

O Eclipse AI foi criado para estudar inteligência artificial local, desenvolvimento de assistentes virtuais e integração de modelos de linguagem com aplicações de terminal.
